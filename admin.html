🔥 Firebase Setup

Connect your website to Firebase so all admin changes reflect live for every visitor — no coding needed.

Step-by-Step Setup Guide

1
Create a Firebase Project
Go to console.firebase.google.com → Click Add project → Name it rehabplus → Disable Google Analytics (optional) → Click Create project
2
Add a Web App
Inside your project → Click the </> (Web) icon → Register app as rehabplus-web → Copy the firebaseConfig object shown
3
Enable Firestore Database
Left sidebar → Firestore Database → Create database → Choose Start in test mode → Select region (asia-south1 for India) → Click Enable
4
Paste Your Credentials Below
Fill in the values from the firebaseConfig object you copied in Step 2

Enter Firebase Credentials

Security Rules (paste into Firestore)

Copy these rules → Firebase Console → Firestore → Rules tab → Replace existing → Publish

⚠️ For the admin to write, you will also need to enable Firebase Authentication (Email/Password) and sign in from admin.html. Or keep "test mode" (allows all writes for 30 days) while you are setting up.

📋 Step-by-Step Setup (5 minutes)

1
Create Google Sheet
Go to sheets.google.com → Click + (New spreadsheet) → Name it RehabPlus Data
2
Create 3 Tabs
At the bottom, create three sheets named exactly: Bookings, Enquiries, Reviews
3
Open Apps Script
In your sheet → Extensions → Apps Script → Delete the default code → Paste the code below
4
Deploy as Web App
Click Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Click Deploy → Copy the URL
5
Paste URL Above
Paste the deployment URL in the field above → Click Save & Connect
